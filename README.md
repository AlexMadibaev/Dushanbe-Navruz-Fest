# Festival Map

Статический сайт интерактивной карты фестиваля для публикации на GitHub Pages.

## Структура

- `index.html` — основной single-file интерфейс карты
- `.github/workflows/deploy-pages.yml` — деплой на GitHub Pages через GitHub Actions

## Публикация

После пуша ветки `main` workflow автоматически задеплоит сайт на GitHub Pages.

## Важное ограничение

Скрытый вход в админку через тройной тап по логотипу и пароль в клиентском JavaScript не является настоящей защитой.
GitHub Pages — это статический хостинг, поэтому весь фронтенд-код доступен пользователю в браузере.

Для реальной защищенной админки позже понадобится отдельный backend, авторизация на сервере или отдельная закрытая CMS.
